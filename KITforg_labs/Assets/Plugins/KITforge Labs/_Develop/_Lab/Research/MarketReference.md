## 2) GitHub signal

Popular open-source Unity repos show what devs actively search for outside the store:

### `MyBox` (2.2k+ stars)
Demand for inspector attributes, micro-utilities, and reusable editor helpers.

### `UnityAssetUsageDetector` (2k+ stars)
Strong need for safe reference search, cleanup workflows, and project confidence tools.

### `Kino` (2.4k+ stars)
Focused, demo-friendly visual effects still attract interest when setup is clear and results are immediate.

## 3) Reddit / public opinion signal

Recurring "must-have" mentions and pain points:

- hierarchy and inspector ergonomics (`vHierarchy`, `vInspector`, `Editor Console Pro`)
- safe cleanup and reference tracking
- faster level building and spline workflows
- in-game debug panels and capture tools
- ready-to-use settings UI and utility inspectors
- URP-friendly visual upgrades that do not require deep graphics knowledge

## 4) What the audience appears to value

1. **Immediate payoff** — value visible in minutes, not days.
2. **Safety** — tools that avoid breaking scenes, references, or assets.
3. **Low setup friction** — ideally works right after import.
4. **Good examples** — demo scene, GIF/video, quick README.
5. **Clear scope** — one problem, one promise.
6. **Professional polish** — stable UI, good naming, no console noise.

## 5) Opportunity filters for KITforge labs

A candidate product is stronger when it is:

- useful across many Unity projects
- easy to explain in one sentence
- demoable in under 30 seconds
- supportable without endless custom setup
- valuable for both solos and small teams
- clearly better, safer, or simpler than current options

## 6) Initial market thesis

`KITforge labs` should not start with giant frameworks.

The best opening portfolio is likely:
- workflow safety tools
- editor productivity tools
- practical capture / export helpers
- focused URP shaders with strong visual demos

That matches the studio site positioning: **practical tools for repeated tasks, awkward steps, and day-to-day production work that should feel easier.**
