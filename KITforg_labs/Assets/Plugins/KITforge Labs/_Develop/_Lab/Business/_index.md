## Structure rationale

```
_Lab/Business/         ← REFERENCE (the "why" and the data)
Skills/KF_MarketRules  ← RULES (the actionable "what to do")
```

The Business docs explain context, market data, and reasoning.
KF_MarketRules extracts the binary rules for AI session consumption.
