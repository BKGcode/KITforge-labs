# README — WaterKIT
**KITforge Labs** · Unity 6 · URP

> Production-ready stylized mobile water. Presets. One price. No extensions.

---

## Requirements
- Unity 6000.0.60f1 or later
- Universal Render Pipeline (URP) 17.0+
- Render Graph enabled (Compatibility Mode must be OFF)
- Platforms: Android, iOS, PC, macOS, WebGL 2.0

## Quick Start
1. Import WaterKIT — the Setup Window opens automatically
2. Confirm your URP is configured (the window checks this for you)
3. Select a water mesh in your scene
4. In the Inspector → **Presets** tab → pick a preset → **Apply**
5. Done. Your water is live.

## Full vs Lite
- **Full** — all features enabled. Targets mid-range Android (Samsung A54) and above.
- **Lite** — depth, foam, normals, translucency, flat shading, river mode. Targets low-end Android (Samsung A14).
- Switch between Full and Lite in the **Performance** tab. All parameter values are preserved.
